⚡ Quick Diagnostics

💻 Computer is slow
1. Open Task Manager → High CPU/RAM?
2. Check storage (low free space causes slowdown)
3. Restart system
• Check temperatures (overheating = throttling)
• Disable startup programs
• Scan for malware
• Consider hardware limits (RAM/CPU/drive age)
🌐 Internet not working
1. Try another website or app
→ One fails = service issue

2. Test another device on same network
→ Only your device = device problem
→ All devices = network or ISP

3. Restart your device
4. Check Wi-Fi or cable connection
Low-risk checks:
• Restart router (if all devices affected)
• Move closer to router (signal issue)
• Check ISP outage
Advanced (only if confident):
• Check IP address status
• Adjust DNS settings
• Reset network configuration
📱 App crashing
1. Restart app/device
2. Check for recent updates or bugs
• Clear cache/data
• Reinstall app
• Check compatibility
• Look for known issues
🔥 Overheating
1. Ensure airflow is not blocked
2. Close intensive apps
• Clean dust from vents
• Check fan operation
• Avoid soft surfaces
• Advanced: replace thermal paste

🧠 Guided Troubleshooting

Start diagnosing your issue
Step 1: Does restarting fix it?
→ Yes: temporary software issue
→ No: continue
Step 2: Is it one app or everything?
→ One app: app-specific problem
→ System-wide: continue
Step 3: Any heat, noise, or lag spikes?
→ Yes: possible hardware issue
→ No: likely software/config issue
Step 4: Did it start after a change?
→ Yes: undo/update/rollback
→ No: deeper diagnosis needed
