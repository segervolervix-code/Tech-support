Device won't power on
Hardware
1. Hold the power button for 10 seconds.
2. Check the power cable and outlet.
3. Remove accessories and try again.
Wi‑Fi connected but no internet
Networking
1. Restart your router.
2. Flush DNS: ipconfig /flushdns.
3. Check if other devices have the same issue.
Software keeps crashing
Software
1. Update to the latest version.
2. Clear the application cache.
3. Check system logs for error codes.
Overheating issues
Thermals
1. Ensure vents are not blocked.
2. Clean dust from fans.
3. Update firmware for thermal fixes.
